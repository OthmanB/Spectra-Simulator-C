----- From this, we get minimum/maximum values for the noise parameters ----
Values of p are often considered to be either 2 (granulation/macrogranulation) or 4 (activity)
But there is a strong degeneracy between p/tau and A, so that fixing it does not change 
the features of the power spectrum in overall (not in the range of the pulsations)
