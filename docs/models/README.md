# Model manuals

This index covers the non-obsolete models (those with examples in `Configurations/examples_cfg/`, excluding `Configurations/examples_cfg/obselete/`). For legacy models, see `docs/legacy_models.md`.

You can also list supported models directly from the CLI:

```bash
./build/specsim --list-models
./build/specsim --describe-model <name>
```
