Function that extracts the relevant information from the table of models.

This function extracts the requested row from the input table of models. The data should be arranged by rows, with the first column containing the model index. The function also requires the frequency files to be in a specific format: freq.[Model_index]. For example, freq.000101.24.

Parameters
tableThe input table that contains information from the models.
table_indexThe column index that is requested to extract.
dir_freqsThe directory that contains all of the frequency files for each model.
Returns
A structure of type Model_data containing the extracted model parameters and frequencies.
Note
The function assumes that the table is in the format specified in the input conditions.
The function uses the Model_data structure defined in data.h.
